# 🚀 빠른 시작 가이드

**처음 사용하시는 분을 위한 3분 가이드**

---

## ⚡ 초간단 설치 (처음 한 번만)

### 1️⃣ Python 설치

1. https://www.python.org/downloads/ 접속
2. 노란색 다운로드 버튼 클릭
3. 다운로드된 파일 실행
4. ⚠️ **중요!** "Add Python to PATH" 체크박스 반드시 체크!
5. "Install Now" 클릭

### 2️⃣ 자동 설치 실행

1. `powder_inspection` 폴더 열기
2. **`install.bat`** 파일 **더블 클릭**
3. 자동으로 설치 진행 (1~2분 소요)
4. "설치 완료!" 메시지 확인

✅ 설치 끝!

---

## 🎯 서버 시작 (매일)

### 방법 1: 더블 클릭 (추천!)

1. `powder_inspection` 폴더 열기
2. **`start.bat`** 파일 **더블 클릭**
3. 검은 창이 나타나면 서버 실행 중!

### 방법 2: 명령어

```bash
cd C:\powder_inspection
python app.py
```

---

## 🌐 접속하기

### 같은 PC에서

- 웹 브라우저 열기
- 주소창에 입력: **`http://localhost:5000`**

### 다른 PC에서 (직원들)

1. 서버 PC의 IP 주소 확인 (서버 실행 메시지에서 확인)
   ```
   * Running on http://192.168.1.100:5000
                      ↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑↑
                      이 주소를 사용!
   ```

2. 직원 PC 웹 브라우저에서:
   ```
   http://192.168.1.100:5000
   ```

## ❓ 문제 발생 시

### "python을 인식할 수 없습니다"
→ Python 재설치 (PATH 체크 확인)

### "ModuleNotFoundError: No module named 'flask'"
→ `install.bat` 다시 실행

### 다른 PC에서 접속 안됨
→ README.md의 "방화벽 설정" 참조

### 기타 문제
→ README.md 파일 참조 또는 IT 담당자 문의
